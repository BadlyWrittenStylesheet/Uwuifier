# Instalation
Git clone repository
=======
##### Git clone repository
```
git clone https://github.com/HlibShutov/Uwuifier.git__
```
##### Install [pynvim](https://github.com/neovim/pynvim) 
```
pip3 install pynvim
```
##### Install uwuipy
```
pip install uwuipy
```

### Packer

```

```
